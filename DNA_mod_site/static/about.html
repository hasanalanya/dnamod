About

Covalent DNA modifications have been found in numerous organisms and more are continually being discovered and characterized, as detection methods improve. Many of these modifications can affect the conformation of the DNA double helix, often resulting in downstream effects upon transcription factor binding. Some of these modifications have been demonstrated to be stable, while others are viewed as merely transient.

DNAmod catalogs information on known DNA modifications, of which the well-known 5-methylcytosine is only one. It aims to profile modifications' properties, with diagrams and links to databases such as ChEBI. It also captures citations to published literature on these modifications. This makes it easier for those new to the field to explore the array of potential modifications.

DNAmod is comprised of this static website and a backing SQLite database. The database is created using Python, including the SOAP client suds and Biopython. The website is also created using Python, makes use of Open Babel via its Python wrapper, Pybel, and uses the Jinja2 templating engine.

This project is open source and its code is on BitBucket. We update DNAmod on a monthly basis and manually curate the modifications verified to occur in vivo.

Methods:

DNAmod uses NCBI’s Chemical Entitles of Biological Interest (ChEBI) database to import potential covalent DNA modifications. It initially imports (via ChEBI Web Services) all children of any of the nucleobases, as indicated by ChEBI’s ontology (specifically, any entry which has functional parent ∈ {A, C, G, T, U}).

These putative modifications are then filtered using ChEBI’s star-based rating system and only modifications which have been assigned three stars (indicating the highest level of manual curation) are retained. Citation data for references provided by ChEBI (via PubMed IDs) are imported using the Biopython Entrez package. This information is then imported into the SQLite database.

To create individual webpages that make up this static website, the SQLite database is imported into Python and sorted through Jinja2 templates. The chemical structure images seen on the modification pages are created by converting ChEBI-provided simplified molecular-input line-entry system (SMILES) information into vector images, via Pybel.

The webpages are demarcated into "Verified" and "Unverified" categories. Assignment to the verified category indicates that the modifications therein were (manually) deemed likely DNA modifications; all other webpages are placed into the unverified category, pending curation or novel functional insights.
